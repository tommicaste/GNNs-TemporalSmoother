In order to train and test the two models using the project data and configurations only two steps must be followed: 

1. Downaload the data from 
https://anwar-said.github.io/anwarsaid/neurograph.html

2. Set path = 'your path' in main

3. Run main

If you want to use your own data the format must be a list of databatches in pytorch geometric format and can be direclty fed into train fucntion
